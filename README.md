# lavamatic

NASA Space Apps Challenge 2015 project
https://2015.spaceappschallenge.org/project/lavamatic/

## Analysis

* Init the venv

```
virtualenv --no-site-packages env
```

* start the analysis notebook:

```
source env/bin/activate
ipython notebook --pylab inline
```

* pip dependencies:

```
use freeze.txt
```
